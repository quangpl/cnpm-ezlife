## Installation
```
git clone [https://github.com/quangpl/cnpm-ezlife](https://github.com/quangpl/cnpm-ezlife)

npm install

npm start
```
Enter link : http://localhost:3000

## Built With

 - [ExpressJS](https://expressjs.com/)
 - Database : MongoDB
 - Bootstrap
 - HTML, CSS and Jquery

## Features

 - Book Manager
 - Staff Manager
 - Account Manager
 - Login / Register
 - Recent Books
 - Cart Feature
